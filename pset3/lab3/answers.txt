sort1 uses: Bubble sort

How do you know?: It's the fastest program when sorting sorted numbers. 
While dealing with sorted numbers, bubble sort is theta(n), merge sort is theta(nlgn), 
selection sort is theta(n^2).

sort2 uses: Merge sort

How do you know?: It's the fastest program when sorting random numbers. 
Merge sort is O(nlgn) while others are O(n^2).

sort3 uses: Selection sort

How do you know?:  It's the slowest program when sorting sorted numbers.
